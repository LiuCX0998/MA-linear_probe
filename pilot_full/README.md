# Data

The full data files are not stored in this GitHub repository because they are large generated experiment outputs.

The data archives are available from Google Drive:

https://drive.google.com/drive/folders/1DxDOlwfLu6A2XeGBITGlF2t8w4n3YPdO?dmr=1&ec=wgc-drive-%5Bmodule%5D-goto

## Expected files

The Drive folder contains the following pipeline outputs:

- `step1_gt-*.zip`: Step 1 ground-truth scene extraction outputs
- `step2_rel-*.zip`: Step 2 extracted relation and cluster outputs
- `step3_div-*.zip`: Step 3 generated text / paragraph outputs
- `step4_probe-*.zip`: Step 4 probe sample outputs
- `step6_probe-*.zip`: Step 6 probing result outputs
- `step7_vis-*.zip`: Step 7 visualization outputs
